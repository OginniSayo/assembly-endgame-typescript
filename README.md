# Assembly Endgame
Save the world of programming from using only assembly to code by playing this hangman-type game.
Very fun and easy to play, with different levels to test your skills.

## Installation Instructions
- Clone the github repository

`git clone https://github.com/OginniSayo/assembly-endgame`

- Open up the folder of the repository and navigate to the client folder as follows

`cd client`

- Install the packages 

`npm install`

- Start the server

`npm run dev`

And enjoy playing around with the program.
